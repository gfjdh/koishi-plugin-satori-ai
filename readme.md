# 觉bot的ai对话

使用前在[https://platform.deepseek.com/](https://platform.deepseek.com) 中获取api-key

对于部署者行为及所产生的任何纠纷， Koishi 及 koishi-plugin-satori-ai 概不负责。

如果有更多文本内容想要修改，可以在 本地化 中修改 zh 内容

# 使用方法

### 指令如下：
| 功能 | 指令 |
|  ----  | ----  |
| 对话 |  sat |
| 清空所有回话 | sat.clear |
| 添加常识 |  sat.common_sense |
